# Resume Analyzer (AI)

An AI-powered Resume Analyzer built using Python and Ollama.

## Tech Stack

- Python
- Ollama
- Llama 3.2
- Prompt Engineering

## Current Features

- Read resume from text file
- Analyze resume using a local LLM
- Return structured JSON
